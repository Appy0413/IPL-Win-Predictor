## Key Insights
- Mumbai Indians and Chennai Super Kings dominate IPL history
- Toss has minimal impact on match outcomes (~50%)
- Venue plays a significant role in chase success
- IPL scores have increased over time

## Features Used
- Target score
- Team win rates
- Venue chase percentage
- Toss decision
- Season normalization
- Recent form (last 5 matches)

## Model
- Logistic Regression (chosen for interpretability)

## Performance
- Accuracy: ~60%
- ROC-AUC: ~0.62

## Key Learnings
- Feature engineering is more important than model complexity
- T20 cricket is inherently unpredictable
- Probabilities are more useful than binary predictions

- ## 🎯 Why This Project?

This project answers a real-world question:
"Given a first innings score, how likely is the chasing team to win?"

It is designed to simulate how data scientists present insights to decision-makers in sports analytics.


## 🔍 Key Insights

- Toss has minimal impact (~50% win rate)
- Venue significantly affects chasing success
- IPL scores have increased over time
- Higher targets reduce win probability

- ## 📉 Model Performance

- Accuracy: ~60%
- ROC-AUC: ~0.62

This reflects the high uncertainty in T20 cricket rather than model weakness.
